# Riley Lynch — The 5 Pillar Method Funnel

Landing page for the "How to Make it D1 & Pro" webinar.

## Deploy

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import from GitHub
3. Select this repo — Vercel auto-detects static HTML, hit Deploy
4. Done. Live in ~30 seconds.

## Before Going Live

- Replace the countdown `TARGET` date in the `<script>` at the bottom of `index.html`
- Replace `onclick="alert('Add checkout link')"` on all three pricing buttons with your real checkout URLs
- Swap the VSL placeholder with your actual video embed (YouTube, Vimeo, or Wistia iframe)
